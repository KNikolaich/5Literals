# WordChainsGenerator

Консольное приложение на .NET 10, которое генерирует цепочки слов так, чтобы во всей цепочке все буквы были уникальными и покрывали максимально возможное количество букв русского алфавита.

Каждой букве русского алфавита назначен вес, соответствующий её частоте употребления в русском языке. Результирующие цепочки отсортированы по убыванию суммарного веса (значимости букв в цепочке).

## Сборка и запуск (Windows PowerShell):

```powershell
# Перейти в папку проекта
cd .\WordChainsGenerator

# Построить (файл words.txt автоматически скопируется в каталог Debug или Release)
dotnet build -c Release

# Запустить с ограничением на длину цепочки (опционально)
dotnet run --project .\WordChainsGenerator.csproj -c Release -- .\words.txt 5

# Параметры: [путь_к_файлу_слов] [макс_длина_цепочки]
```

## Формат результата

Выводятся цепочки слов, отсортированные по убыванию веса:
- **Слова в цепочке** (через запятую)
- **Букв**: количество уникальных букв, покрытых цепочкой
- **Вес**: сумма частот этих букв (чем выше, тем более значимые буквы в цепочке)

Пример вывода:
```
Максимально покрыто букв: 25/33
Лучшие цепочки (отсортированы по весу букв в порядке убывания):

полно, изюбр, чешуя, мышца, дрейф
  Букв: 25, Вес: 0,9279
```

## Входные данные

Формат `words.txt`: одна строка — одно слово (UTF-8, только русские буквы). Слова с повторяющимися буквами внутри будут автоматически проигнорированы.

## Технические детали

- Целевая платформа: .NET 10.0
- Русский алфавит: 33 буквы (включая ё)
- Веса букв основаны на статистике частоты букв в русском языке
- Алгоритм: DFS с битовыми масками и отсеиванием ветвей для оптимизации
- При сборке файл `words.txt` автоматически копируется в выходной каталог (`bin/Release/net10.0` или `bin/Debug/net10.0`) только если был обновлён
