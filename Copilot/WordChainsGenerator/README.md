# WordChainsGenerator

Консольное приложение на .NET 10, которое генерирует цепочки слов так, чтобы во всей цепочке все буквы были уникальными и покрывали максимально возможное количество букв русского алфавита.

Сборка и запуск (Windows PowerShell):

```powershell
# Перейти в папку проекта
cd .\WordChainsGenerator

# Построить (файл words.txt автоматически скопируется в каталог Debug или Release)
dotnet build -c Release

# Запустить
dotnet run --project .\WordChainsGenerator.csproj -c Release -- .\words.txt 5

# Параметры: first arg - путь к файлу со словами (UTF-8), второй arg (опционально) - макс. длина цепочки
```

Формат `words.txt`: одна строка — одно слово. Только русские буквы. Слова с повторяющимися буквами внутри будут проигнорированы.

**Примечание**: при сборке файл `words.txt` автоматически копируется в выходной каталог (`bin/Release/net10.0` или `bin/Debug/net10.0`) только если он был обновлён с момента последней сборки.
